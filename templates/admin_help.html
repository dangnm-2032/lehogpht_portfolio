Cách Add/Edit Work:

*Thao tác trên máy tính hoặc điện thoại*

1. Tạo thư mục mới trên máy

2. Kéo thả các file ảnh vào thư mục

3. Tạo file meta.yaml trong thư mục

4. Copy nội dung sau vào file meta.yaml

name: #Tên dự án client: #Ai thuê year: industry: #Eg FMCG - Household & Home Care whatwedid: #Eg Visual Identity, Pakaging title: #Tiêu đề description: | #Mô tả dự án roi1_number: #Có thì ghi roi1_desc1: roi1_desc2: roi2_number: roi2_desc1: roi2_desc2: roi3_number: roi3_desc1: roi3_desc2: hero_image: #Điền tên file, vd DaBacXanh.png images: [ # Điền tên file [ĐBX_KeyVisual 2.png], [DaBacXanh.png, ĐBX_KeyVisual 2.png] ]

5. Zip folder và upload lên web
